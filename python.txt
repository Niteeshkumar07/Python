                            ********** Slicing **********

-- Fetching or extacting a part of a collection
-- It is supported by those data types who supports indexing concept

                        var_name/coll[si : ei+-1: updation/step_value]

si(starting index) --> From where should i start. (default value is 0)
ei(ending index) --> Where should i stop?(exclusive)
updation/step-value --> How to go to the next one ? (+1)

                +1 --> Positive Index --> Left --> Right

                -1 --> negative Indexing : Left <----- Right


                                        >>> s = 'Hello Python'
                                        >>> s[::1]
                                        'Hello Python'
                                        >>> s[0:4]
                                        'Hell'
                                        >>> s[0:5]
                                        'Hello'
                                        >>> s[0:5:1]
                                        'Hello'


                                        >> l=[1,2,3,4,5]
                                        >>> l[-5:-3:1]
                                        [1, 2]
